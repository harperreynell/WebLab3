## Home page

Home page is showing small description of what is user looking at. Also, home page is presenting most popular tools right now.

## Tools page

Tools page is presenting a full list of tools. User can search for tools in the special search field on the left side of the page. Each tool is in the card with a name of the tool, par of the code snippet and tags. When clicking on the card, representing the tool, user sees a tool details with full code snippet, detailed description, link to github and link to similar tools.

## About page

About page is showing detailed idea of the project, github link to the project repository and a link to the feedback form.
